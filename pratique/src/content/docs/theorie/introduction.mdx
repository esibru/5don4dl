## concepts clés

* Scaling vertical (up) et horizontal (out) (système distribué)
* Base - ACID
* CAP
* Normalisation et Dénormalisation
* Polyglot
* Schéma flexible
* Support des jointures

## Modèles de données

* Modèle orienté document (MongoDB)
* Modèle orienté clé-valeur (Redis)
* Modèle orienté famille de colonne (HBase, Cassandra, Hypertable)
* Modèle orienté graphe (neo4j)
* Modèle orienté Série chronologique (InfluxBD)

## Absence de schéma

Mythe ou ralité -> Nous parlerons plutôt de schéma à la lecture ou schéma implicite. Quelles sont les conséquences ?

* Manque d'optimisation
